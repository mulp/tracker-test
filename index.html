Tracker Blocking Test

This page tests whether tracker blocking is working.

How to verify tracker blocking:

  1. Note how many trackers are blocked with protection ON (shield icon is green)
  2. Click the shield icon to turn protection OFF (icon turns red)
  3. Reload the page and compare the results
  4. You should see fewer trackers blocked when protection is OFF

Note: Some trackers might still show as blocked even with protection off due to browser security policies.
